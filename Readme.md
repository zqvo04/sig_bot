# 무빙워칭 (Moving Watching)

OKX 선물 시장 대상 암호화폐 트레이딩 시그널 봇.  
BTC/USDT · ETH/USDT · HYPE/USDT · SOL/USDT의 15분봉 기준 롱/숏 진입 신호를 텔레그램으로 전송합니다.

---

## 아키텍처

```
GitHub Actions (15분 스케줄)
    │
    ├─ data_pipeline.py        OKX API 데이터 수집
    ├─ microstructure_analyzer.py  마이크로구조 분석 (6개 방안)
    ├─ analysis_engine.py      기술 지표 + SMC 분석
    ├─ scoring_system.py       점수 산출 + 신호 판정
    ├─ notification.py         텔레그램 메시지 발송
    └─ config.py               전역 설정 파라미터
```

## 점수 계산 공식

```
최종점수 = (raw_score × EMA배율 × gate_penalty + 보너스_합계) × soft_penalty + micro_penalty

raw_score     = Σ (지표점수 × 국면별가중치)      0 ~ 100pt
EMA배율       = 역방향 TF 수에 따른 억제 배율
gate_penalty  = 펀딩비·롱숏비율 동시 불리 시 0.80
보너스_합계   = Σ 보너스 항목 (캡 적용 후)
soft_penalty  = MTF RSI 패널티 × EXPLOSIVE 소진 × 캔들모멘텀 × CHoCH
micro_penalty = 마이크로구조 6개 방안 패널티 합산 (최대 −30pt)
```

### ADX 제거 이유 (v3)
EMA 배율이 이미 방향 불확실성을 반영합니다. ADX를 별도 배율로 추가하면 `EMA×0.72 × ADX×0.85 = ×0.61` 형태의 이중 억제가 발생합니다. ADX는 **국면 분류**[...]

## 시장 국면 분류

| 국면 | 조건 | 신호 임계값 |
|------|------|------------|
| RANGING | MA20 교차 ≥2회 or ER < 0.35 | **60pt** |
| TRENDING | ADX ≥ 25, 추세 진행 중 | **62pt** |
| EXPLOSIVE | ADX ≥ 40 + BB 확장 1.2× | **58pt** |
| SQUEEZE | BB 스퀴즈 + ADX < 25 | **63pt** |

SIGNAL_MIN_SCORE = 63pt (모든 국면 공통 하한)

## EMA 배율 테이블

EMA 역방향 TF 수(0~3)에 따라 원점수를 억제합니다.

| 역방향 TF 수 | RANGING | TRENDING | EXPLOSIVE | SQUEEZE |
|:-----------:|:-------:|:--------:|:---------:|:-------:|
| 0 (전부 동방향) | ×1.00 | ×1.00 | ×1.00 | ×1.00 |
| 1 | ×0.96 | ×0.88 | ×0.93 | ×0.95 |
| 2 | ×0.90 | ×0.72 | ×0.84 | ×0.87 |
| 3 (전부 역방향) | **×0.82** | **×0.52** | ×0.75 | ×0.80 |

> **RANGING 완화 이유**: 박스권에서는 EMA가 반전 신호를 오억제합니다.  
> RSI 극단 과매도 + BB 하단이탈 상황(클라이맥스 셀링)에서도 신호를 포착할 수 있도록 0.82로 완화했습니다.

base 점수(EMA 배율 적용 후)에 따라 보너스 상한이 결정됩니다.

| base 점수 | 보너스 최대 | 비고 |
|:---------:|:-----------:|------|
| < 36pt | 18pt | 중립 시장 — 보너스로 구제 불가 |
| 36~44pt | 26pt | 약한 방향성 |
| ≥ 44pt | **36pt** | 명확한 방향성 |

최종 가능 점수 상한: `100 × 1.0 + 36 = 100pt`  
(기존 35pt → **36pt**로 완화 — 강한 신호 포착 강화)

## 페널티 체계

### Soft 패널티 (배율 방식)

| 조건 | 배율 |
|------|:----:|
| MTF RSI 극단 과매수(롱) / 과매도(숏) — 강 | ×0.85 |
| MTF RSI 극단 과매수(롱) / 과매도(숏) — 약 | ×0.92 |
| EXPLOSIVE 소진 (1h RSI 극단 과열) | ×0.88 |
| 연속캔들 모멘텀 역방향 (RANGING) | ×0.80 |
| 연속캔들 모멘텀 역방향 (TRENDING) | ×0.90 |
| CHoCH 역방향 진입 (구조 전환 경고 중) | ×0.88 |

### 마이크로구조 패널티 (절대값, 최대 −30pt)

| 방안 | 최대 패널티 | 조건 |
|------|:-----------:|------|
| 청산 캐스케이드 감지 | −15pt | API 실시간 청산 데이터 |
| 오더북 즉시 벽 | −12pt | 현재가 ±0.3% 내 대형 벽 |
| OI 속도 반대방향 | −15pt | SHORT_SQUEEZE/LONG_LIQ |
| BB 방향 부적합 | −18pt | RANGING에서 역방향 BB 극단 |
| 마크가격 괴리 | −14pt | Last vs Mark >0.5% 이탈 |
| LS 괴리(고래) | −10pt | 계좌수 vs 포지션수 >15%p 차이 |

### 기타 억제

- **BB streak 억제**: TRENDING에서 BB 하단(상단) 3캔들 이상 연속 이탈 시 롱(숏) 완전 차단
  - 예외: 15m RSI < 28 (극단 과매도) → 클라이맥스 셀링으로 판단, 억제 해제
- **OI Spike 필터**: OI 급증(≥80%) + Taker 역방향 → −20pt

## 극단 과매도/과매수 처리

전 타임프레임 RSI가 동시에 극단적일 때의 특수 처리:

```
판정 조건 (롱):
  15m RSI ≤ 32 AND 1h RSI ≤ 32 AND 4h RSI ≤ 35
  AND BB 상태: lower_breakout / near_lower / lower_zone

효과:
  1. 멀티TF 극단과매도 보너스: +10pt
  2. BB streak 억제 면제: 15m RSI ≤ 28 → TRENDING에서도 롱 허용
```

**설계 의도**: 하락 막바지 클라이맥스 셀링 구간에서는 EMA가 당연히 모두 bearish입니다.  
RANGING 3역방향이어도 ×0.82 배율이 적용되어 반등 신호 포착이 가능합니다.
